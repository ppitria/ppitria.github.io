Online Retail Sales Exploration using SQL (Dummy Data Project)

Eksplorasi data penjualan ritel untuk memahami performa transaksi dan perilaku pelanggan menggunakan SQL.


Proyek ini merupakan latihan eksplorasi data menggunakan SQL untuk memahami penggunaan query dalam proses analisis sederhana. Fokus utamanya adalah menulis dan mengoptimalkan perintah SQL seperti JOIN, GROUP BY, dan fungsi agregasi (SUM, COUNT, AVG) untuk menghasilkan informasi dasar seperti total transaksi, pendapatan, pelanggan teraktif, dan distribusi penjualan berdasarkan lokasi.
Walaupun proyek ini bersifat eksploratif, prosesnya membantu saya memahami penggunaan SQL sebagai alat dalam pengolahan dan interpretasi data mentah. Melalui proyek ini, saya belajar menyusun query yang efisien, menjaga konsistensi struktur database, serta memperoleh insight dasar dari data hanya melalui kode SQL.


Tools: SQL

    Dari proyek ini saya belajar:
  • Membuat query SQL untuk analisis deskriptif seperti total transaksi, pendapatan, dan perilaku pelanggan.
  • Menggunakan JOIN untuk menggabungkan beberapa tabel transaksi, pelanggan, dan produk.
  • Memanfaatkan GROUP BY, HAVING, dan fungsi agregasi untuk analisis berbasis kelompok
  • Menulis query yang efisien, terstruktur, dan mudah dibaca dengan penggunaan alias dan komentar.
